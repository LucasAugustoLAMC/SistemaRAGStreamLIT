 # Suporte Inteligente com RAG + Groq + Streamlit

Um **agente de suporte ao cliente com inteligência artificial** completo, rápido e 100% funcional — feito com Retrieval-Augmented Generation (RAG), Groq (velocidade insana), FastEmbed, ChromaDB e interface web em Streamlit.

Funciona 24h por dia, responde com base na sua base de conhecimento e escala para produção.

## Funcionalidades

- Respostas instantâneas com Groq (Llama 3.1, Kimi, etc.)
- Busca semântica com FastEmbed + ChromaDB (persistente em disco)
- Interface web bonita e responsiva com Streamlit
- Totalmente em português (PT-BR)
- Fácil de customizar com sua base de conhecimento
- Zero configuração complexa
